支持

需要StrideQuest的帮助？请查看以下常见问题或直接联系我们。

联系方式

请发送邮件至 stridequests.support@gmail.com，我们会尽快回复。

常见问题

如何连接Apple Health？

在初始设置期间，StrideQuest会请求从Apple Health读取步数和步行距离的权限。如果您跳过了此步骤，请前往设备的设置 → 健康 → 数据访问与设备 → StrideQuest并启用访问。本应用仅读取数据——绝不会向Apple Health写入数据。

我的步数没有显示

请确保Apple Health权限已启用（见上文）。StrideQuest在启动时会导入最近几天的步数数据。尝试关闭并重新打开应用。如果您使用第三方步数追踪器，请确保它与Apple Health同步。

如何启用或禁用通知？

您可以在应用内的设置 → 提醒中配置步行提醒。要完全禁用通知，请前往设备的设置 → 通知 → StrideQuest并关闭允许通知。

如何取消订阅？

订阅通过Apple管理。前往设备的设置 → Apple ID → 订阅，找到StrideQuest，点击取消订阅。您的高级访问权限将持续到当前计费期间结束。

如何恢复购买？

打开StrideQuest，前往设置 → 订阅，点击恢复购买。这将与App Store同步您的订阅状态。请确保使用购买时相同的Apple ID登录。

我的健康数据是否私密？

是的。您的健康数据存储在设备本地，绝不会被上传、出售或共享。详情请参阅我们的隐私政策。

StrideQuest Premium包含什么？

Premium解锁免费Moonlit Meadow旅程之外的额外旅程、伙伴风格自定义和扩展功能。提供月度和年度订阅。

重新安装后丢失了进度

应用进度存储在设备本地。如果您重新安装应用，一些偏好设置可能通过iCloud恢复，但健康衍生的进度（每日目标、能量、旅程状态）存储在本地，可能无法恢复。我们建议保持应用安装以保留您的数据。
